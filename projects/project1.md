# Project 1 - Boolean Logic

## Project Description
In this project, I built fundamental logic gates just using **Nand** gates.
I used the HDL (Hardware Description Language) provided by the nand2tetris course.

After building all these gates by hand, I understand that I can built so complex curcuits
just by using a single universal gate (Nand or Nor).

## Implemented Components
* Not
* And
* Or
* Xor
* Mux
* DMux
* Not16
* And16
* Or16
* Mux16
* Or8Way
* Mux4Way16
* Mux8Way16
* DMux4Way
* DMux8Way

## Key Concepts Learned
* The **Nand gate** is a universal gate capable of implementing any Boolean function.
* Hardware design follows a hierarchical approach, where higher-level components are
constructed from lower-level building blocks.
* HDL provides a structural description of digital circuits rather than a sequence of instructions.

## Skills Practiced
* Digital logic design
* Boolean algebra
* Combinational circuit design
* Hardware Description Language (HDL)
* Hierarchical hardware construction
